Adminsitracion Tienda

Explicación:

Aplicación, para gestionar el catalogo de una tienda de videojuegos, su stock, realizar ventas, y devoluciones de las compras. Todo mediante un menu de terminal.

Al iniciar el programa muestra el menu.

Desde aquí se pueden elegir que acción se quiere hacer.

Opción 1.

Muestra el catalogo de juegos, con sus Códigos de compra, y el precio correspondiente.

Opción 2.

Mustra el listado de Stock que tiene la tiene antual mente.

Opción 3.

Muestra el listado de Stock de menos a mas.

Opción 4.

Muestra el catalogo, ordenado por el precio de mayor a menor.

Opción 5.

Con esta opción podemos realizar compras, se calculara el precio de la compra, y proporcionara el ticket.

Comprobación del Stock por la venta.

Como se puede ver, se han restado las cantidad que se han comprado al Stock de los juegos.

Opción 6.

Esta Opción sirve para devolver una compra entera, actualizando el Stock, para que se puedan volver a comprar esos productos.

Comprobación del Stock por la devolucion.

Aquí vemos que vuelve a tener el Stock que tenia antes de la compra.

Opción 7.

Finalizamos la ejecución, y se cierra el programa.
